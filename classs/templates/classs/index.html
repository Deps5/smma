Query the database using Python requests

The most common operations(no idea if thats the technical term) you will perfrom on a database is get data from it and post data to it.

Below is some python code showing how to do both of these tasks:




import requests #this is the module which comes built into Python which we will use make get and post requests

url = "http://smma.pythonanywhere.com/students" #create a string of the url

response = requests.get(url) #use a get request passsing he url as an argument

response = response.json() #turn it into a json object(which will be a normal list of dictionaries in python)

print(response) # print the json data out toi the console

payload = {"name":"Some random year 7", "age":"12", "experience":"0.1"} #here we create a dictionary containing the data we would like to post to the database

response = requests.post(url, data=payload)#use a post request to post the data
